# blockcodingVR-unity
# VR 소프트웨어 코딩 교육 플랫폼

------
4차 산업혁명 시대, 필수요소가 된 코딩 교육을 위한 VR 블록코딩 시스템입니다.
학습모드는 VR 블록코딩과 게이미피케이션 요소가 적용되었습니다.
멀티플레이에서는 샌드박스를 활용하여 직접 맵을 만들어 교수자-학습자 간 비대면 언택트 VR 블록코딩 교육이 가능합니다.
